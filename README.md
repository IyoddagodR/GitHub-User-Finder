## Описание программы

**GitHub User Finder** — это десктопное GUI-приложение для поиска пользователей GitHub с использованием официального GitHub API. 

Программа позволяет быстро находить пользователей по логину или имени, просматривать основную информацию и сохранять интересных пользователей в **избранное**. Все избранные пользователи сохраняются в JSON-файл и доступны после перезапуска программы.

### Основные возможности:
- Поиск пользователей GitHub в реальном времени
- Отображение логина, имени, количества подписчиков и репозиториев
- Добавление пользователей в избранное (двойной клик)
- Сохранение избранного в `favorites.json`
- Проверка на пустое поле поиска
- Удобный графический интерфейс на Tkinter

---
Как пользоваться

Запустите приложение.
Введите имя или логин пользователя в поле поиска (например: torvalds, octocat, google).
Нажмите кнопку "🔍 Искать" или нажмите Enter.
В результатах поиска дважды кликните по нужному пользователю.
Подтвердите добавление в избранное.
Нажмите кнопку "❤️ Избранное", чтобы посмотреть сохранённых пользователей.

Валидация: При пустом поле поиска появляется предупреждение.
